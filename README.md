## 📌 專題簡介
**緣來如此** 是一個結合「緣分匹配」與「月老解籤」的月老緣分分析系統。  
系統會根據雙方資料與當日星象生成個人化戀愛建議，並可以誠心求籤以線上解籤，得到白話文結果。

🎯 **專題定位**
結合 **AI 技術 × 傳統月老文化**，為單身大學生打造兼具娛樂與實用性的戀愛指引系統
---
### 🔧 核心功能
* 💑 **AI 緣分配對**：星座 × 興趣 × MBTI × Love API
* 🧧 **AI 月老籤詩解籤**：傳統籤詩 → 白話戀愛建議
---
### 🧠 使用技術
* **Python**（requests / pandas / BeautifulSoup）
* **Gemini AI**（配對分析、籤詩現代化詮釋）
* **Google Sheets**（籤詩與歷史紀錄）
* **Gradio**（互動式介面）
---
### ✨ 亮點特色
* AI × 傳統文化的創新應用
* 配對結果與籤詩「互相呼應」
* 每次皆為 **個人化生成結果**
